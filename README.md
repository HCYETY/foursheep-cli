## 使用

在计算机上已经装好 Node 开发环境后，全局安装 `foursheep-cli` ：

```shell
npm install -g foursheep-cli
```

使用 foursheep-cli 创建一个新项目：

```shell
foursheep init <项目名称>
cd <项目名称>
npm install
```
